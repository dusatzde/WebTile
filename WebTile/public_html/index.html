Web Live Tile

Vítejte na stránkách projektu Web Live Tile. Díky tomuto projektu můžete připnout své oblíbené webové stránky na home screen vašeho telefonu. Pokud nenaleznete ikonu vašeho webu v Top section, můžete vytvořit vlastní odkaz v Custom section.

Podporován je Windows Phone 7 i Windows Phone 8. K vytvoření custom web live tile stačí zadat URL, vybrat categorii a připnout vygenerovanou stránku na plochu. Služba je zdarma a bez reklam. Enjoy!
